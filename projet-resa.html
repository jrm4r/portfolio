Présentation du projet

Description

Le projet RESA a été réalisé dans le cadre du BTS SIO SLAM. Il s'agit d'une application de gestion de réservations permettant aux utilisateurs de réserver des ressources (salles, équipements…) via une interface simple et intuitive.

L'application gère les conflits de créneaux, les droits d'accès selon le rôle de l'utilisateur, et offre un suivi complet de chaque réservation : création, modification et annulation.

Objectifs atteints
