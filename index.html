Cloud & Architecture 2024 – 2025
L'évolution des technologies serverless

Veille réalisée sur le modèle serverless et son impact croissant sur les architectures cloud et microservices, dans le cadre de la formation BTS SIO SLAM.

Définition

Le serverless est un modèle de développement cloud dans lequel les développeurs exécutent du code sans gérer l'infrastructure serveur. La gestion des serveurs, de la mise à l'échelle et de la maintenance est entièrement assurée par les fournisseurs de services cloud.

Plateformes majeures
☁️
AWS Lambda
Amazon Web Services
🔷
Azure Functions
Microsoft Azure
Cas d'usage
  • Exécution de code en réponse à une requête HTTP
  • Déclenchement à l'ajout d'un fichier en stockage cloud
  • Traitement de messages dans une file d'attente
